## Project Overview
- Form2PDF is a React-based web application built with TypeScript and Material-UI that allows users to create dynamic forms and convert them to PDF documents. It's currently in early development stages with a foundation for form building functionality.

## Technology Stack
- Frontend Framework: React 19.2.0 with TypeScript
- Build Tool: Vite with SWC for fast refresh
- UI Library: Material-UI (MUI) v7.3.6 with icons and date picker components
- Date Handling: Day.js for date operations
- Development: ESLint for code quality, TypeScript for type safety
